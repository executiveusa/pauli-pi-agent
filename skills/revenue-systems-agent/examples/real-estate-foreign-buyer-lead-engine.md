# Example: Real Estate / Foreign Buyer Lead Engine

## System Name

Foreign Buyer Trust & Lead Routing System

## One-Line Pitch

We help Mexico real estate businesses capture and qualify serious foreign buyers, build trust before the first call, and route each lead to the right verified partner.

## Funnel

```text
Foreign Buyer → Trust Page → Buyer Intake → Qualification → Verified Partner Routing → WhatsApp Follow-up → Referral Tracking → Closed Deal/Testimonial
```

## Core Screens

1. **Buyer Landing Page** — Trust-building copy, testimonials, FAQ, simple CTA
2. **Mexico Buyer Trust Checklist** — Answers common foreign buyer fears (taxes, legal, process, financing)
3. **Buyer Intake Form** — Budget, timeline, location, property type, financing status
4. **Verified Agent/Partner Directory** — Searchable list of trusted agents, builders, notaries, lenders
5. **Lead Routing Dashboard** — Agent sees new leads, qualification level, contact info
6. **WhatsApp Follow-up Queue** — Templated messages sent at day 1, 3, 7
7. **Referral Tracker** — Commission owed per agent, lead source attribution

## Key Automations

- Send buyer guide on signup
- Route lead to matching agent
- Alert partner immediately
- Follow up after 1/3/7 days
- Log referral source
- Ask buyer if they connected
- Request testimonial after close

## Subscription Fit

**Premium or Enterprise.** One closed real estate referral can justify the monthly cost. High margin business.
