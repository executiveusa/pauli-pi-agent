# Example: Destination Wedding / Events / Retreat Engine

## System Name

Destination Event Lead & Package Coordination System

## One-Line Pitch

We help Puerto Vallarta wedding, event, and retreat businesses capture serious high-ticket inquiries, qualify them fast, and coordinate trusted local vendors through one simple system.

## Funnel

```text
Event Lead → Inquiry Page → Budget/Date/Guest Count Qualifier → Planner Dashboard → Vendor Matching → Package Proposal → Deposit → Review/Referral
```

## Core Screens

1. **Event Inquiry Landing Page** — Hero, testimonials, portfolio, "Get Started" CTA
2. **Guest Count/Budget Qualifier** — "How many guests?" "What's your budget?" "When?"
3. **Package Builder** — Venue + catering + decor + transport + photography combos
4. **Vendor Matching Dashboard** — Recommended vendors based on event type
5. **Planner Dashboard** — New inquiries, qualification level, next steps
6. **Client Presentation Page** — Custom proposal with chosen vendors and pricing
7. **Follow-up Tracker** — Payment milestone, vendor confirmations, pre-event checklist
8. **Referral/Commission Tracker** — Commission owed per vendor, lead source

## Key Automations

- Send planning checklist on inquiry
- Notify planner/venue immediately
- Follow up after 1/3/7 days
- Recommend vendor bundle based on event type
- Send deposit/payment link
- Collect review after event
- Trigger anniversary follow-up (renewal booking)

## Subscription Fit

**Premium or Enterprise.** One wedding (avg $30k–$100k budget) justifies the system several times over.

## Pauli Sent Me Fit

**Very high.** This can become a local trust and package marketplace. Every vendor (photographer, caterer, florist, transportation) becomes a Pauli Verified partner with tracked referral commission.
