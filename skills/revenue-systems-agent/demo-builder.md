# Demo Builder

## Purpose

Create a demo spec that can be built quickly and used for outreach.

The demo does not need to be a full production app.

It needs to show the owner:

1. how the system captures money
2. how the system saves time
3. how the system builds trust
4. how the system simplifies operations
5. why the monthly subscription makes sense

## Demo Types

| Demo Type | Use When |
|---|---|
| Static mockup | Fast outreach |
| Clickable prototype | Stronger sales call |
| Branded dashboard | Best for serious prospects |
| Live mini-app | Best for niche template product |
| Video walkthrough | Best for cold outreach |

## Required Demo Screens

Every demo should include at least:

1. Customer-facing landing/intake page
2. Owner dashboard
3. Lead/client profile
4. Follow-up or automation screen
5. Reporting/value screen

## Optional Screens

- referral tracker
- package builder
- quote builder
- document checklist
- WhatsApp queue
- partner dashboard
- presentation mode
- customer status page
- review capture page

## Demo Spec Format

```json
{
  "demo_name": "",
  "niche": "",
  "business_result": "",
  "screens": [
    {
      "screen": "Owner Dashboard",
      "purpose": "Show new leads, status, and follow-up actions"
    }
  ],
  "automations": [],
  "data_model": {},
  "build_complexity": "low | medium | high",
  "recommended_demo_type": "static | clickable | live-mini-app | video-walkthrough"
}
```
