# Phased Execution Checklist

## Phase 1 — Intake + Normalize
- [ ] Pull canonical state from Notion.
- [ ] Normalize to ProjectUnit schema.
- [ ] Attach lane and ownership metadata.

## Phase 2 — Prioritize + Decompose
- [ ] Score with revenue/mission/reuse/risk weights.
- [ ] Select overnight-eligible items.
- [ ] Decompose into bounded tasks with rollback path.

## Phase 3 — Execute + Verify
- [ ] Run assigned task in isolated surface/worktree.
- [ ] Capture command transcript and evidence links.
- [ ] Run required checks/tests.

## Phase 4 — Distill + Report
- [ ] Persist episodic memory entry.
- [ ] Promote candidate procedural patterns.
- [ ] Publish summary and unresolved escalations.
