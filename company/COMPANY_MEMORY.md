# COMPANY_MEMORY.md - Memory Layers & Continuity

We operate across 8 structured memory layers, curated by the Memory and Knowledge Agent:

1. **Company Memory**: Vision statement, core models, operating budgets, and directories.
2. **Human Preferences**: Bambu's feedback, guidelines, custom styles, and approval logs.
3. **Client Memory**: Specific brand details, business hours, and translation preferences.
4. **Product Memory**: Synthia design templates and Smart Site layout tokens.
5. **Agent Memory**: Teammate roles, tool permissions, and active session histories.
6. **Workflow Memory**: Scopes, intake YAML parameters, and QA checklists.
7. **Skill Memory**: The `skills.md` file, tracking modular external tool adapters.
8. **Market Memory**: Prospect criteria, high-quality reviews, and niche analysis.

Every evening, the Memory Agent summarizes active sessions, prunes stale database assumptions, and graphs decisions onto our PostgreSQL pgGraph database.
