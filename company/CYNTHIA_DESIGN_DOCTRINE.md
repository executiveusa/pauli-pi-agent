# CYNTHIA_DESIGN_DOCTRINE.md - Luxury Minimalism

The Synthia design doctrine rejects cheap generic AI layouts. It enforces high-end editorial and spatial visual hierarchies.

## Aesthetic Pillars

### 1. Luxury Minimalism
Controlled whitespace, generous paddings, clean grid proportions, and Outfit-Sans high-end typography.

### 2. Atmospheric Depth
Sleek dark mode using coordinated HSL color palettes. Subtle gradient glassmorphic panels (`backdrop-filter`) and drop-shadow definitions that make UI components feel spatial.

### 3. Cinematic Motion
Smooth micro-animations on hover states, gentle fades, and transitional loading states to make the interface feel alive and premium.

## The UDEC Score Gate
Before any client deliverable or UI is published, the Cynthia Design Director calculates the User Experience Design (UDEC) rating. The checklist evaluates responsiveness, type scaling, alignment, speed, and contrast.
* **Nothing ships below a UDEC score of 8.5/10.**
