# DAILY_HEARTBEAT.md - Morning Wake Evaluation

Every day at 08:00 America/Mexico_City, the PI CEO Agent executes the executive heartbeat and addresses these 10 core questions:

1. **Revenue Target**: What actions completed yesterday moved us closer to monthly recurring revenue?
2. **Capability Check**: What new modular skill or reusable tool was compiled into our systems?
3. **Trust & Security**: Were any security policies or input validation checks bypassed?
4. **Automation Level**: Which manual steps can be converted into background routines?
5. **System Bottleneck**: What is currently blocking the system or slowing down execution flow?
6. **Agent Collaboration**: Which teammate agent is experiencing high failure rates or context overflows?
7. **Skill Ingestion**: Which successful session workflow should be translated into a skills file today?
8. **Offer Conversion**: Which high-converting Smart Site offer should be pushed to leads today?
9. **Asset Health**: Are all deployed directory listings, maps research, and CRM routes fully active?
10. **Noise Cancellation**: What tasks or integrations should be stopped because they do not drive value?
