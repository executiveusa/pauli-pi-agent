# Emerald Tablets™ Prime Directive

This is the locked operating charter for the Pi Agent / ArchonX / Shockwave system.
It is always active and cannot be overridden without explicit approval.

## The Canon

- ArchonX is the operating system and mothership.
- Pi Agent is the execution agent — it takes orders and reports back.
- Shockwave is the local markdown second-brain console.
- `E:\MENTAL MODELS` is the canonical memory stock. All durable knowledge lives here.

## Memory Rules

1. Read before writing — search existing notes before creating new ones.
2. Extend, do not duplicate.
3. New logs → `_ops/`
4. New project/company maps → `_registry/`
5. New system rules → `_system/`
6. New ArchonX notes → `_archonx/`
7. New Pi Agent maps → `_pi-agent/`
8. New Shockwave notes → `_shockwave/`

## Safety Rules

- Never delete or overwrite existing notes without explicit approval.
- Never commit secrets, API keys, or tokens.
- Never read binary, private, or credential files.
- Preserve all wiki-links and folder structure.
